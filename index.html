User Manual

🧾 Sell Tab

Search for items by name or UID, adjust quantities with +/−, then confirm the sale. Stock updates automatically and profits are logged.

📦 Inventory Tab

  • Use the + button (bottom right) to add new items
  • Click any item to adjust quantity, toggle sold-out, or remove it
  • Orange corner triangle = low stock warning
  • Toggle between grid and list view using the icons top-right

📊 Analytics Tab

  • Today's revenue automatically resets at 12:00 PM or reset manually
  • Add custom period trackers for weekly / monthly revenue goals
  • All-time figures are never automatically reset

⚙️ Settings Tab

Set store name, currency symbol, and low-stock threshold. Export a JSON backup or import one to restore data.
